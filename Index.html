I am a Game Design and Development major at MSU, and I love creating interactive experiences that feel intuitive, polished, and fun. Featured project

Drifter game prototype

A Unity-based 3D drifting game focused on precision handling, smooth drift mechanics, and real-time scoring. I created custom physics for controlled drifting, tuned traction and steering systems, and designed UI elements such as speed, RPM, and drift score feedback. The game features multiple vehicles, responsive controls, and a polished environment, allowing for a satisfying drifting gameplay loop while exploring vehicle dynamics and core gameplay systems.
